# 우물 안 개구리

우물 밖을 향해 나아가세요!

다양한 아이템을 먹어 능력을 사용하세요!

수 많은 능력으로 당신의 앞을 가로막는 장애물을 뛰어넘으세요!

당신은 어디까지 올라갈 수 있을까요?

조작 방법 :
  - 이동 : WASD
  - 아이템 먹기 : F
  - 능력 사용하기 : E
